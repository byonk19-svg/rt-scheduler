# CONTEXT.md Implementation Matrix

This matrix tracks the product contract in `CONTEXT.md` against the current codebase. Status values:

- `Implemented`: protected by code, schema, or tests.
- `Partial`: behavior exists, but a rule is not fully enforced or visible.
- `Missing`: no durable implementation found.
- `Needs browser proof`: source/tests suggest support, but the workflow still needs role-based browser validation.
- `Deferred`: intentionally left out of this implementation pass with a concrete reason.
